# Package index

## Import

Functions to import raw spectra, identifications or clusters

- [`import_biotyper_spectra()`](https://clavellab.github.io/maldipickr/reference/import_biotyper_spectra.md)
  : Importing spectra from the Bruker MALDI Biotyper device
- [`import_spede_clusters()`](https://clavellab.github.io/maldipickr/reference/import_spede_clusters.md)
  : Import clusters results generated by SPeDE
- [`read_biotyper_report()`](https://clavellab.github.io/maldipickr/reference/read_biotyper_report.md)
  : Importing Bruker MALDI Biotyper CSV report
- [`read_many_biotyper_reports()`](https://clavellab.github.io/maldipickr/reference/read_many_biotyper_reports.md)
  : Importing a list of Bruker MALDI Biotyper CSV reports

## Process

Functions to analyze and verify multiple spectra data

- [`check_spectra()`](https://clavellab.github.io/maldipickr/reference/check_spectra.md)
  : Evaluate the spectra regularities

- [`merge_processed_spectra()`](https://clavellab.github.io/maldipickr/reference/merge_processed_spectra.md)
  : Merge multiple processed spectra and peaks

- [`process_spectra()`](https://clavellab.github.io/maldipickr/reference/process_spectra.md)
  :

  Process Bruker MALDI Biotyper spectra *à la* Strejcek et al. (2018)

- [`remove_spectra()`](https://clavellab.github.io/maldipickr/reference/remove_spectra.md)
  : Remove (raw or processed) spectra

## Dereplicate

Functions to cluster and reduce redundancy

- [`delineate_with_identification()`](https://clavellab.github.io/maldipickr/reference/delineate_with_identification.md)
  : Delineate clusters from taxonomic identifications
- [`delineate_with_similarity()`](https://clavellab.github.io/maldipickr/reference/delineate_with_similarity.md)
  : Delineate clusters from a similarity matrix

## Cherry-pick

Function to pinpoint and label specific spectra within clusters

- [`pick_spectra()`](https://clavellab.github.io/maldipickr/reference/pick_spectra.md)
  : Cherry-pick Bruker MALDI Biotyper spectra
- [`set_reference_spectra()`](https://clavellab.github.io/maldipickr/reference/set_reference_spectra.md)
  : Set a reference spectrum for each cluster

## Miscellaneous

- [`is_well_on_edge()`](https://clavellab.github.io/maldipickr/reference/is_well_on_edge.md)
  : Identify the wells on the plate's edge
- [`gather_spectra_stats()`](https://clavellab.github.io/maldipickr/reference/gather_spectra_stats.md)
  : Aggregate spectra quality-check statistics
- [`get_spectra_names()`](https://clavellab.github.io/maldipickr/reference/get_spectra_names.md)
  : Extract spectra names and check for uniqueness
